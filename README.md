AihaBot
=======

This is a multi-purpose local bot that I created for a brazilian anime server that I manage.<br>
You can find it [here](https://discord.com/invite/zz6zFS5).

Info
----
Current Features:
* Fun
* Moderation
* Leveling system

Future:
* Music
* Economy
* GIFs & Images API
* Image editing
